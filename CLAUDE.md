# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

# Project Overview

This repository is used for research and data analysis projects, primarily written in Stata.

# Code Style

- Language: Stata
- Use descriptive variable names
- Keep functions small and focused on a single task

# File Organization

- Raw data files must not be modified — always work on copies
- Store outputs (charts, reports, processed data) in an `outputs/` folder
- Keep scripts in a `scripts/` folder

# Notes

- Prioritize reproducibility — code should produce the same results every time
